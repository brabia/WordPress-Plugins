Earn money by making your Wordpress articles shorter 

== Installation ==

This section describes how to install the plugin and get it working.   
1- WHAT I NEED TO USE WP Url Shortener
<ul>
<li>login</li>
<li>api key</li>
</ul>  
2- WHAT I SHOULD DO  
<ul>
<li>As soon as you install and activate WP Url Shortener a new menu appears at the bottom of your menu list [ Url Shortener ]</li>
<li>You should put your login and api key, get them from https://bitly.com/</li>
<li>A new column should apear's on your post list table with the name [ Short Url ]</li>
<li>Start sharing your short url and earn extra money</li>
<li>Enjoy !</li>
</ul>
