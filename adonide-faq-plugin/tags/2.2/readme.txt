== Installation ==
= Install from WordPress Dashboard =
	1. In your Admin, go to menu Plugins > Add
	2. Search for `HTML FAQ PAGE`
	3. Click to install
	4. Activate the plugin
= Install from FTP =
	1. In your Admin, go to menu Plugins > Add
	2. Select the tab "Upload"
	3. Upload the .zip file you just downloaded
	4. Activate the plugin
