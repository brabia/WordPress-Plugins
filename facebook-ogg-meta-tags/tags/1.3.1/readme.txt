== Description ==
Social Meta Tags will generate social media meta data for all new posts that you publish. 
This allows to share the right titles, descriptions, images and more when your posts are shared into Twitter, Facebook.

== Installation ==

- In your Admin, go to menu Plugins > Add
- Search for `Social Meta Tags`
- Click to install
- Activate the plugin

== Frequently Asked Questions == 

= Why Social Meta Tags plugin does not appear's any where on my admin area? =

Social Meta Tags will not add any menu on your admin area, you can check Social Meta Tags works on your page code source [ CTR+U ]
